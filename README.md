## docker lnmp一键环境 ##
> 说明：内置php7.2 redis4.0 mysql5.7 memcached mongodb 开箱即用
>1. git clone 项目
>2. 进入项目目录 sudo docker-compose up --build -d
>3. 等待10分钟左右即可 访问localhost可以查看效果

### 目录说明
>1.web目录为代码存放目录
>2.app目录下存放了各类服务的配置和数据存放
>3.source 目录存放php扩展编译的源码文件或目录
